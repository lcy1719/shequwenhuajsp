### 万字文档+社区文化宣传网站+SSM+JSP
## 项目采用技术：
开发环境：IDEA、JDK1.8、Maven、MySQL
技术栈：Java、MySQL、Spring、SpringMVC、Mybatis/MybatisPlus、JSP、JavaScript、CSS

系统功能模块：
本系统包含管理员、用户2个角色。
管理员角色：
1. 用户管理：管理系统中所有用户的信息，包括查看、修改和删除用户。
2. 社区新闻管理：管理社区新闻信息，包括添加、修改和删除新闻。
3. 社区公告管理：管理社区公告信息，包括添加、修改和删除公告。
4. 社区活动管理：管理社区活动信息，包括添加、修改和删除活动。
5. 医疗服务管理：管理医疗服务信息，包括添加、修改和删除服务。
6. 社区美食管理：管理社区美食信息，包括添加、修改和删除美食。
7. 家政服务管理：管理家政服务信息，包括添加、修改和删除服务。
8. 活动报名管理：管理活动报名信息，包括查看、修改和删除报名。
9. 家政预约管理：管理家政预约信息，包括查看、修改和删除预约。
10. 系统管理：管理系统配置，如轮播图管理等。
用户角色：
1. 个人信息管理：管理自己的个人信息，包括用户名、密码、用户姓名、用户手机等。
2. 活动报名管理：查看和删除自己报名的活动信息。
3. 家政预约管理：查看和删除自己的家政预约信息。
